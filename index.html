Installation

This section describes how to install the plugin and get it working.

  1. Download plugin archive, unpack plugin directory
  2. Upload plugin directory to the /wp-content/plugins/ directory
  3. Activate the plugin through the 'Plugins' menu in WordPress
  4. Place <?php $data = faker_generate('name', 'name'); ?> in your scriptis

Frequently Asked Questions

How to get current instance \PHPFaker\Faker?

Current Faker instance available in Faker::getFaker() static method

How can I create a generator?

You must extend the class \PHPFaker\Generator\AbstractGenerator and register it in faker method \PHPFaker\Faker::registryGenerator(). After that, the generator can be obtained from an current Faker instance or faker_generate('my_generator', '...') function.
